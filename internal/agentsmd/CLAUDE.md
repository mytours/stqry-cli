# CLAUDE.md

This project manages STQRY content through the `stqry` CLI.

Before doing any stqry work, load both of these skills:

- `/stqry-reference` - commands, flags, and the data model.
- `/stqry-workflows` - step-by-step recipes for multi-step content tasks.

If the slash commands are not available, install them with:

    stqry setup claude --global
